Subarray Sum Equals K

Medium

Pattern: Prefix sum + hash map.

The key idea is that a subarray sum can be found by subtracting two prefix sums. While scanning the array, I store how many times each prefix sum has appeared. If current_sum - k appeared before, then there is a subarray ending at the current index with sum k.

Complexity: O(n) time, O(n) space.

from collections import defaultdict

class Solution:
    def subarraySum(self, nums: list[int], k: int) -> int:
        prefix_count = defaultdict(int)
        prefix_count[0] = 1

        current_sum = 0
        result = 0

        for num in nums:
            current_sum += num
            result += prefix_count[current_sum - k]
            prefix_count[current_sum] += 1

        return result

What I learned: This problem looks like a normal nested-loop subarray problem, but the real trick is representing each subarray as the difference between two prefix sums.
