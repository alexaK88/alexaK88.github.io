About Me

I am a software engineer focused on backend systems, real-time data pipelines, and machine learning workflows. My main work is in industrial software, where raw process data needs to become reliable, usable information for monitoring and decision-making.

I have worked with FastAPI, OPC UA, Pandas, Streamlit, Grafana, Prometheus, Docker, and ML models for process monitoring and prediction. I enjoy building systems that are not only technically interesting, but also useful in messy real-world environments.
