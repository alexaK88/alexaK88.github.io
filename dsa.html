Two Sum

Easy

Pattern: Hash map / complement lookup.

The simple way would be to check every pair of numbers, but that would be slow. Instead, I keep a hash map of numbers I have already seen. For each number, I calculate the value needed to reach the target. If that needed value is already in the hash map, I found the answer.

For example, if the target is 9 and the current number is 2, I need 7. So I check whether 7 was already seen.

Complexity: O(n) time, O(n) space.

class Solution:
          def twoSum(self, nums: list[int], target: int) -> list[int]:
              seen = {}
      
              for i, num in enumerate(nums):
                  complement = target - num
      
                  if complement in seen:
                      return [seen[complement], i]
      
                  seen[num] = i

What I learned: A hash map lets me avoid checking every possible pair. Instead of asking “which two numbers add up to the target?”, I ask “what number do I need to complete the pair?”
